# Security Policy

## Reporting a Vulnerability

セキュリティ上の問題を発見された場合は、Issue で公開せず以下までご連絡ください。

- Email: `contact@bittercute.dev`
- GitHub Security Advisories: このリポジトリの **Security** タブから **"Report a vulnerability"**

3 営業日以内に一次返信いたします。

## Scope

- このリポジトリのコード
- デプロイされた https://bittercute.dev

## Out of Scope

- 第三者サービス (Cloudflare, GitHub 等) の脆弱性は各サービスへ直接ご報告ください
